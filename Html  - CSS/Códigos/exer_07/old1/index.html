Usando links externos

Você pode acessar o meu repositório público no GitHub

Você também pode acessar o meu canal gratuito do youtube

Ao usarmos a tag <a> temos atributos em torno dela como href, target, rel, hreflang,download e type.
Quando utilizar os temos atributos temos valores denominados a eles como:
  1. Para o atributo href
    2. Onde colocamos o link do site que vai ser indexado

  2. Para o atributo hreflang
    3. Onde decidiremos a tradução da página

  3. Para o atributo target
    4. _blank - vai abrir em outra página o hiperlink
    _self - vai abrir na mesma página o hiperlink
    Normalmente usamos mais os valores _blank e _self

  4. Para o atributo rel
    5. Podemos indicar qual é a natureza do hiperlink que clicamos:
    next - indicará que o link é para próxima parte do link atual
    prev - indica que o link é para parte anterior do link atual
    author - indica que é um link para o site do autor
    external - indica que é um link que não faz parte desse site
    nofollow - indica que é um link pago

  5. Para o atributo download
    6. Bom parece bem simples não? Para o atributo download faremos com que baixermos o link ou arquivo.

  6. Para o atributo type
    7. Para complementarmos o atributo download faremos com que o type o complemente. Os valores estão no site IANA.org.
